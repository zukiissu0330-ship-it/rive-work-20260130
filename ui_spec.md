# UI Spec（このテンプレのコンポーネント定義）

## 目的
- 提示デザイン（添付）を「構造＋トークン」で再現し、使い回し可能なテンプレにする。

## トークン
- 変更は `tokens/tokens.css` のみを基本線（色・文字サイズ・フォント・余白・角丸・レイアウト幅）。

## コンポーネント一覧（G2入力の `components`）
1. `topbar`
   - 左: ブランド（名称＋サブ）
   - 右: 住所＋電話（2行）
2. `hero`
   - フル幅（コンテナ内）画像
3. `lead`
   - オレンジの説明文（中央）
4. `gallery` / `lightbox`
   - 3列グリッド（SPは1列）
   - 画像クリックで拡大表示＋テキスト
5. `free`
   - タイトル（オレンジ）
   - 2カラムのフリーテキスト（SPは1カラム）
6. `shop`
   - 左: 店舗情報
   - 右: 決済/QRなどのバッジ画像
7. `social`
   - SNSアイコン（丸・枠あり）
8. `map`
   - Google Maps埋め込み

## 承認条件（G2）
- コントラスト: ブランド背景 `--color-brand` 上の文字が視認できること（実測は Lighthouse / aXe で確認）。
- 可読: 本文 `--font-size-m`（16px）以上、行間 `--line-height-base`=1.7。
- 操作: リンク・ボタンのフォーカス可視（必要なら `:focus-visible` を追加）。
