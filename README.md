# Sirius-like Reusable Template（G2成果物）

## 使い方
1) `index.html` をブラウザで開く  
2) 色・フォント・文字サイズ・余白などは `tokens/tokens.css` を編集  
3) 画像は `assets/` の差し替えでOK

## ローカル確認（Hero 16:9）
- `npm` は不要。`index.html` をブラウザで開く
- 画面幅を変更して、Heroが常に16:9枠・中央基準のトリミング表示になることを確認

## ファイル
- `tokens/tokens.css` …… デザイン・トークン（ここだけ触るのが基本）
- `styles/base.css` …… ベース（リセット/コンテナ）
- `styles/components.css` …… 各UI部品
- `ui_spec.md` …… コンポーネント仕様
