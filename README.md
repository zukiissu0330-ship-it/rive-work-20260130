# Sirius-like Reusable Template（G2成果物）

## 使い方
1) `src/pages/index.html` をブラウザで開く  
2) 色・フォント・文字サイズ・余白などは `src/tokens/tokens.css` を編集  
3) 画像は `src/assets/` の差し替えでOK

## ファイル
- `src/tokens/tokens.css` …… デザイン・トークン（ここだけ触るのが基本）
- `src/styles/base.css` …… ベース（リセット/コンテナ）
- `src/styles/components.css` …… 各UI部品
- `ui_spec.md` …… コンポーネント仕様
